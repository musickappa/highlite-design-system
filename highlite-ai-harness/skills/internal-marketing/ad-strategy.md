## Highlite 固定コンテキスト（毎回使う情報）

### サービス概要
- **提供価値**: AIを使ったデザイン機能を社内に内製化するサポート（3ヶ月〜）
- **差別化**: 「ツール導入で終わり」ではなく、自走できる体制・フローまで設計して伴走
- **LP**: https://highlite.co.jp/lp/quartzsetup/
- **CTA 2本立て**: `無料ヒアリングを予約する`（今すぐ相談層）/ `まず資料を見てみる`（情報収集層）

### ターゲット（2軸）

| ターゲット | 定義 | 刺さるペイン |
|---|---|---|
| **① デザインに困っている経営者** | 外注依存・内製化できない・コストと時間がかかる | 「外注費が積み重なる」「修正に1〜2週間かかる」 |
| **② SIDに危機感を感じている経営者** | SaaSモデルの限界に気づき、ブランド・体験の差別化が急務 | 「プロダクト力だけでは戦えなくなってきた」 |

### 推定企業像
- 業種: SaaS・自社プロダクト系IT・スタートアップ
- 規模: 従業員10〜50名（専任デザイナー不在 or 1名）
- フェーズ: PMF後〜グロース期
- 意思決定者: CEO・COO・事業責任者

## 媒体選定ガイド

| 判断軸 | Googleリスティング | Meta広告 |
|---|---|---|
| 向いている状況 | 「デザイン内製化」「AI導入」など検索意図がある層に即リーチしたいとき | ターゲット①②を広くリーチして認知を積みたいとき |
| コンバージョン速度 | 速い（検索意図ありき） | 遅い（認知→関心→行動のステップが必要） |
| 向かない状況 | 検索ボリュームが少ないキーワード領域 | 短期で問い合わせを刈り取りたい場合 |
| 推奨フェーズ | 今すぐ獲得が必要な短期目標 | 中長期のリード育成・リターゲティング |

**原則**: 短期目標（月1件獲得など）→ Googleリスティング優先。認知・中長期 → Meta。両立時は予算比率 7:3（Google:Meta）を起点に調整。

## Google リスティング広告 設計フロー

### 1. キャンペーン構造

```
キャンペーン: Highlite_[ターゲット名]_Search
  └── 広告グループ: [訴求軸ごとに分ける]
        └── キーワード + 広告文
```

**ターゲット①向け（デザインに困っている経営者）**
```
広告グループ候補:
- 内製化検討層:    デザイン 内製化, 社内 デザイン 体制
- コスト課題層:    デザイン 外注 コスト削減, デザイン 費用 削減
- AI活用検討層:    AI デザイン 導入, Figma AI 活用
- スピード課題層:  デザイン 対応 遅い, デザイン 外注 時間かかる
```

**ターゲット②向け（SID文脈）**
```
※ SID（SaaS is Dead）のキーワード定義が確定後に追記
```

### 2. キーワード設計

各広告グループで以下3種を組み合わせる:

| マッチタイプ | 用途 | 例 |
|---|---|---|
| 完全一致 `[  ]` | コンバージョン刈り取り | [デザイン 内製化 支援] |
| フレーズ一致 `" "` | 関連検索を広く拾う | "デザイン 体制 構築" |
| 除外キーワード | 無関係クリックを防ぐ | 求人, 転職, 無料, 独学, ツール 使い方 |

**必須除外キーワード（初期設定）**:
```
求人 / 転職 / バイト / 副業 / スクール / 学習 / 独学 / 資格 / 無料ソフト
個人 / フリーランス / ポートフォリオ
```

### 3. 広告文（RSA形式）

RSA（レスポンシブ検索広告）で見出し15本・説明文4本を用意し、Googleが自動で最適組み合わせを学習させる。

**見出し案（ターゲット①向け）**
```
1. デザイン外注から卒業しませんか
2. 3ヶ月で社内デザイン体制を構築
3. 専任デザイナー不要のAI内製化
4. 月30〜50万円の外注費を削減
5. 当日対応できる社内体制をAIで
6. Highlite — デザイン内製化の伴走支援
7. 無料ヒアリング30分で現状診断
8. AI×デザインで社内に余白を生む
9. ブランド一貫性をAIで維持する
10. 外注依存から抜け出す3ヶ月プラン
```

**説明文案**
```
1. ChatGPT・Figma AIを社内に根づかせるフロー設計と伴走支援。ツール導入で終わらせません。
2. デザイン外注に月20〜50万円かけていませんか。3ヶ月で社内完結できる体制をつくります。
3. 「ちょっと修正」が即日対応できる社内体制を、AI活用と仕組み設計でゼロから構築します。
4. 才能は不要。「このAIにこう指示すれば良い」という型を社内スタッフに定着させます。
```

### 4. ランディングページ & CTA 対応

| 検索意図 | 誘導先 | CTA |
|---|---|---|
| 今すぐ相談したい | LP直接 | 無料ヒアリングを予約する |
| まだ情報収集中 | LP → DL資料CTA | まず資料を見てみる |

### 5. 入札・予算設定

```
初期フェーズ（データ収集期）:
- 入札戦略: クリック数最大化（コンバージョン実績が5〜10件貯まるまで）
- 1日予算: 目標獲得件数 × 想定CPA ÷ 30 を基準に設定
- CPC上限: 500〜1,500円（BtoBサービスの相場帯）

データが貯まった後:
- 入札戦略: 目標コンバージョン単価（tCPA）に切り替え
```

## Meta広告 設計フロー

### 1. キャンペーン構造

```
キャンペーン: [目的]_[ターゲット]_[時期]
  ├── 広告セット A: ターゲット① (コールドオーディエンス)
  ├── 広告セット B: ターゲット② (コールドオーディエンス)
  └── 広告セット C: リターゲティング (LP訪問者)
```

### 2. オーディエンス設定

**ターゲット①向け（コールドオーディエンス）**
```
職種ターゲティング:
- 役職: CEO, 代表取締役, 社長, COO, 事業部長
- 業種: インターネット, ソフトウェア, スタートアップ, ITサービス
- 従業員規模: 11〜200名
- 興味関心: Figma, Notion, ChatGPT, スタートアップ, デザイン思考, SaaS
```

**リターゲティング（温オーディエンス）**
```
- LP訪問者（過去30日）
- ページ滞在時間上位25%
- 除外: 既存顧客
```

### 3. 広告クリエイティブ方針

| フォーマット | 用途 | 訴求軸 |
|---|---|---|
| 静止画（1枚） | 認知・クリック | 数字訴求（「月30万の外注費が消える」） |
| カルーセル | 詳細説明 | ビフォー/アフターを各カードで見せる |
| 動画（15〜30秒） | 認知拡大 | 「デザイン外注の典型的な1日」シナリオ形式 |

**コピー軸（ターゲット①）**
```
主テキスト（Primary Text）案:
「デザイン修正を外注に頼んで、返ってくるまで2週間——
その間、商機が動いています。

Highliteは、3ヶ月で『デザインをAIで社内で回せる体制』をつくります。
専任デザイナーは不要。既存スタッフがAIを使いこなせる型を設計します。」

見出し（Headline）案:
「デザイン外注をやめた会社が、次にやったこと。」
「月50万の外注費を、社内の力に変える。」
```

### 4. 予算・スケジュール

```
テスト期（最初の2〜3週間）:
- 1広告セットあたり: 日予算1,000〜2,000円
- 目的: 広告セット・クリエイティブの優劣を判断するデータ収集

スケールアップ（優勝広告セットに集約）:
- 週次で ROAS / CTR / CPL を確認
- 負け広告セットは2週間でオフ判断
```

## 注意・制約

- **予算未定の場合**: 小規模テスト（Google: 日1,000〜3,000円 / Meta: 日1,000〜2,000円）から始めることを前提に戦略設計する
- **ターゲット②（SID文脈）**: 定義・キーワードが未確定のため、設計前にユーザーと確認を取ること
- **DL資料CTA**: LP上にまだ実装されていない（議事録確認済み）。広告文でDL資料への誘導を入れる場合は実装済み前提か確認する
- **実績・社会的証明**: 現状ゼロ。広告文に実績数字や事例を使わないこと
