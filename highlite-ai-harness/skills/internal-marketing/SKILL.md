# Internal Marketing — 発信と関係性の継続

Crystaフェーズで結晶化した価値を、外向きに**静かに広げる**。Ore接点創出の燃料も供給。

## このスキルが扱う範囲

| 役割 | サブスキル | トリガーフレーズ |
|---|---|---|
| 事例記事化 | `case-study.md` | 「事例書いて」「[顧客名]のケーススタディ」 |
| SNS / コンテンツ発信 | `social-post.md` | 「LinkedIn投稿作って」「note書いて」 |
| 広告戦略設計 | `ad-strategy.md` | 「広告の戦略考えて」「リスティング始めたい」「Meta広告どうすればいい」「広告設定教えて」 |

## 共通の作法

1. **必ず参照する**: `../../CLAUDE.md` / `../_shared/brand-voice.md` / `../_shared/glossary.md`
2. **Push先**:
   - 事例化 → `#cry-ナーチャリング・イベント-cc`
   - SNS / コンテンツ発信 → `#ore-リード・アサイン情報-cc`（接点創出はOre側に集約）
3. **顧客名の扱い**: Crysta 01「実績開示の許可」を経たもののみ実名OK。それ以外は `glossary.md` の伏字ルールに従う

## 設計思想

- **発信は売り込みではなく贈り物** — 自社の宣伝より、読者の判断を助ける情報
- **頻度より中身** — 月数本で十分。毎日投稿は目指さない
- **顧客の声を主役に** — Highliteは脇役として登場する
- **「磨き」のプロセスを見せる** — 完成品より、考え方・葛藤・選択を共有
